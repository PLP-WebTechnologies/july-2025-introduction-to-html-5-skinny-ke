Introduction

This simple web page demonstrates how to use semantic HTML5, enhance accessibility for assistive technologies, and apply SEO-friendly practices for better discoverability.

Why Semantic HTML Matters

1. Clear Document Structure

Semantic tags define the role of each section, making content easier to understand for users and machines alike.

2. Improved Accessibility

Using proper landmarks like <nav> and <main> helps screen readers navigate efficiently.

3. SEO Optimization

Search engines use headings and content hierarchy to better understand and rank your page.
